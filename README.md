# Miromap — Save AI Chats as PDF, Markdown & More

> One extension. Six AI platforms. Five export formats.

Export conversations from ChatGPT, Claude, Gemini, DeepSeek, Grok & Perplexity as PDF, Markdown, TXT, JSON, or CSV.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- **Next.js 15** (App Router, Turbopack)
- **Tailwind CSS 4**
- **Framer Motion** (scroll animations)
- **Lucide React** (icons)

## Deployment

Deploy on Vercel:

```bash
npm run build
```

Or connect this repo to Vercel for auto-deploy.
